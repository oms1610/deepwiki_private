## システム機能一覧表（DeepWiki-Open）

| No | 区分 | 機能名 | 概要 | 主な入力 | 主な出力／結果 | 備考 |
|----|------|--------|------|----------|----------------|------|
| F-01 | UI | リポジトリ指定 | 解析対象となるGitリポジトリを指定する | リポジトリURL | 対象リポジトリ確定 | GitHub/GitLab/Bitbucket対応 |
| F-02 | UI | Wiki生成条件設定 | Wiki生成時の各種条件を設定 | 言語、モデルプロバイダ、モデル名、再生成フラグ | 生成条件確定 | LLM選択・言語指定 |
| F-03 | UI | アクセストークン登録 | Privateリポジトリ用のアクセストークンを登録 | PAT(Token) | トークン保存 | UI上で管理 |
| F-04 | UI | Wiki生成実行 | リポジトリ解析とWiki生成を開始 | 生成条件一式 | Wiki生成処理開始 | Generate Wiki |
| F-05 | UI | 生成進捗表示 | Wiki生成中の進捗・ログを表示 | 生成ステータス | 進捗／完了／エラー表示 | ストリーミング更新 |
| F-06 | UI | Wiki閲覧 | 生成されたWikiを閲覧 | ページ選択 | Wikiページ表示 | 目次・階層構造 |
| F-07 | UI | Mermaid図表示 | Wiki内のMermaid図を描画 | Mermaid記法 | 図のレンダリング | 構成図・フロー図 |
| F-08 | UI | Ask（Q&A） | リポジトリ内容に基づく質問応答 | 質問文 | 回答 | RAGベース |
| F-09 | UI | DeepResearch | 複雑なテーマを多段で調査・要約 | 調査テーマ | 調査レポート | 複数ステップ推論 |
| F-10 | UI | エクスポート | Wikiコンテンツを外部出力 | 出力形式指定 | Markdown/JSON等 | ドキュメント再利用 |
| F-11 | API | ヘルスチェック | サービス稼働状態を確認 | なし | 稼働ステータス | /health |
| F-12 | API | Wiki生成API | Wiki生成処理をAPI経由で実行 | repo_url, model設定 | 生成結果 | /wiki/generate |
| F-13 | API | Chat / RAG API | RAG検索＋生成応答 | 質問・文脈 | 回答 | Ask機能用 |
| F-14 | API | モデル管理 | 利用可能なLLM設定管理 | モデル設定 | 利用モデル反映 | 環境変数管理 |
| F-15 | 共通 | 多言語対応 | UIおよび生成Wikiの多言語化 | 言語設定 | 表示／生成言語変更 | i18n |
| F-16 | 共通 | キャッシュ管理 | 生成結果や索引のキャッシュ | リポジトリID | 再利用／高速化 | 再生成制御 |
| F-17 | 共通 | セキュリティ管理 | APIキー・トークン管理 | APIキー | 認証済み通信 | BYOK前提 |
